# weather_report
気象庁の過去のデータをスクレイピングで抽出

# はじめに
PythonのSeleniumを使用した静的サイトの自動操作方法について知るために参考になりそうなものを実際に書いて実行できるか確認してみることにしました。

# 使用技術など
言語：Python
Selenium:サイトの呼び出し、自動操作
Google chrome：今回使用したのブラウザ
